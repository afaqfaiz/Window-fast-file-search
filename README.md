# Fast File Search & Explorer

A high-performance desktop file search application built with PyQt6.

##  Features

- Trigram-based inverted indexing for fast search
- Real-time folder indexing
- Extension filtering (manual & dropdown)
- Built-in file explorer view
- Context menu (copy path / reveal file)
- Cross-platform support (Windows, macOS, Linux)

##  Tech Stack

- Python
- PyQt6
- Threaded indexing system
- Inverted Index Search Algorithm
